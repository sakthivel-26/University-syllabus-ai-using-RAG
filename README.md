# 📖 Overview

The **University Syllabus AI Assistant** is an AI-powered system that allows users to:

- Upload syllabus PDF documents
- Ask questions about course content
- Retrieve answers directly from uploaded documents

The system uses **semantic search + vector embeddings + LLM reasoning** to generate accurate answers grounded in the uploaded syllabus documents.

---

# ✨ Features

- 📄 **PDF Upload** – Upload syllabus documents
- 🔍 **Semantic Search** – Retrieve relevant document sections
- 💬 **AI Question Answering** – Ask questions about course details
- 🚫 **Anti-Hallucination Design** – Answers only use document context
- 💾 **Persistent Storage** – Documents stored in ChromaDB
- 📑 **Source References** – Shows which documents were used
- 🖥 **Interactive UI** – Built using Gradio
- ⚙️ **Configurable Pipeline** – Easily change models and settings
